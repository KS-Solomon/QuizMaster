## 题型说明

| 题型 | type 值 | 说明 |
|------|---------|------|
| 判断题 | `judge` | `answer` 为 `0`（正确）或 `1`（错误），仅 Unit 6 有此题型 |
| 单选题-同义句 | `single` | 4 个长句选项中选与原句含义一致的一项 |
| 单选题-同义词 | `single` | 给出英文释义，从 4 个单词中选匹配项 |
| 单选题-选词 | `single` | 选择正确单词填入句子（Unit 6 only） |
| 完形填空 | `cloze` | 两种格式：共享选项池（6.3, 10.3）和每题独立 ABCD 选项（8.1, 9.2） |

> QuizMaster 已支持完形填空的两种格式自动识别与渲染。

## 使用方法

### JSON 文件（导入刷题）

将需要的 JSON 文件复制到项目 `data/` 目录下即可：

```bash
# 示例：导入 Unit 6 的所有题目
cp 研究生英语/6.1.json data/
cp 研究生英语/6.2.json data/
cp 研究生英语/6.3.json data/
```

也可通过网页端 **录入 → 批量导入** 上传 JSON 文件。

> 建议按单元分题库管理（如 `U6`、`U7`……），方便分单元刷题和统计。

### Markdown 文件（查阅 / AI 生成）

`U6.md` ~ `U10.md` 按单元整理了全部题目，每个题型带完整 JSON 块，可直接发给 AI 助手生成题库，或作为纸质参考答案查阅。

## 答案格式

- **判断题**：`0` = 正确，`1` = 错误
- **单选题**：数字索引（A=0, B=1, C=2, D=3）
- **完形填空**：数字数组，`answer[i]` 表示第 `i` 个空在其选项组中的索引
